HRPulse Java Project

This project simulates an HR system with Employees, Attendance, Payroll, Leave, and Grievances.

How to Run

  1. Download the source code `HRPulse.java`.
  2. Compile: javac HRPulse.java
  3. Run: java HRPulse

Features

  • Employee Management
  • Attendance Tracking
  • Payroll Corrections
  • Leave Approvals
  • Grievance Handling
